# SensorsManager

## Application for getting data from available phone sensors and showing readings on charts.

### Supported sensors:
- Accelerometer
- Gyroscope
- Light
- Linear acceleration
- Magnetic field
- Proximity
- Rotation vector
- Heart rate

### Used dependencies:

- JUnit5
- Mockito
- Koin
- Kotlin Coroutines
- Kotlin Flow
- com.github.PhilJay:MPAndroidChart
- Flurry (for DEBUG)
- Retrofit2 (for DEBUG)
